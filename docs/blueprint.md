# **App Name**: RubricAI

## Core Features:

- Assignment & Rubric Input: User interface for uploading student assignment files (PDF/DOCX/TXT) and providing rubrics (paste text or upload file).
- AI-powered Rubric Parsing Tool: Intelligently converts varied rubric input formats into a structured JSON for consistent evaluation.
- AI-driven Criterion-wise Evaluation Tool: Leverages an LLM to generate criterion-wise scores, detailed feedback, and specific improvement suggestions, avoiding full rewrites.
- AI-powered Originality Indicator Tool: Analyzes the submitted text for repetitive patterns, predictable language, and overly generic phrasing to provide a 'low/moderate/needs review' risk assessment.
- Multilingual Feedback Tool: Dynamically translates generated feedback and improvement suggestions into selected languages (English/Hindi for MVP).
- Structured Feedback Display: A clear, interactive user interface to visualize criterion scores, expand feedback sections, show overall grades, and display originality indicators.

## Style Guidelines:

- Primary color: A deep, professional blue (#2357BA) representing clarity and reliability in academic evaluation. This color ensures strong contrast on a light background.
- Background color: A very light, desaturated grey-blue (#EBEFF3), visibly sharing the primary hue, providing a clean and understated canvas for content.
- Accent color: A vibrant, clear cyan (#31BFE6), analogous to the primary blue, used for highlighting interactive elements and calls to action, ensuring distinctiveness.
- Headline font: 'PT Sans' (humanist sans-serif) for clear and modern titles that convey professionalism with a touch of personality.
- Body font: 'Inter' (grotesque-style sans-serif) for highly readable, neutral body text, optimized for presenting detailed feedback and suggestions.
- Utilize simple, outline-style icons that visually support academic concepts and evaluation processes without distraction, promoting clarity.
- Implement a clean, modular layout with clear information hierarchy. Feedback sections should be easily expandable, and results presented in well-defined tables or lists for quick digestion.
- Subtle animations should guide the user, such as progressive scoring visualizations per criterion and smooth transitions when expanding feedback details or switching output languages.
